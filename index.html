ABOUT

The web application is intended to visualize some properties of the universe according to the closed Friedmann–Lemaître–Robertson–Walker model
(also known as the closed Friedmann model).

The universe is assumed to be the three-dimensional spherical space whose radius changes with time.
This dependency is plotted on the left side of the page.

CONTROLS

Moving your mouse while the left button is down will rotate the camera; the controls are akin to those seen in first-person video games.

You can also control the simulation time using the left and right arrow keys.

You can control the simulation speed using the up and down arrow keys.

Pressing the Space key will pause/unpause the simulation.

Pressing the Backspace button allows turning the camera exactly backward.
